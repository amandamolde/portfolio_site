Brew Review

Review and Discover Breweries

The Project

Brew Review was my capstone for General Assembly. We were tasked with designing a product that we wanted to build and choosing appropriate technologies to build it. I like to visit breweries, but sometimes I don't know which brewery to go to, so I decided to make a brewery review website.

My Role

I tackled this project individually.

Technical Components

React | Django | PostgreSQL | Bootstrap

My focus on this project was to show my abilities in React, so I used Django to get the back-end up and running quickly.

Future Improvements

The ability for users to upload their own photos to a review.
