pastTents

Campsite Review Application

The Project

PastTents was my first full-stack application.

My Role

PastTents was created with help from Cameron Binion, collectively we are known as team CamAnda! I completed the Models, Views, and Controllers for both the Campsites and the Reviews.

Technical Components

Express | Node.js | MongoDB | Bootstrap

Future Improvements

The ability for a user to upload photos with their review. We used a document-oriented database (MongoDB) because it was the only database we knew at the creation of this project. Going forward, I would use a relational database due to the relationships between campsites and their associated reviews. We could also then tie a created review to a user.
