pastTents

HTML, CSS, Bootstrap, JavaScript,
jQuery, Node, Express, MongoDB

PastTents is a campsite review app. It was created with help from Cameron Binion, collectively we are known as team CamAnda! Users can login using their email or with Google. Once logged in, a user can create a new campsite, view a list of created campsites, and view each campsite individually.

A user can also create a review for an existing campsite, view a list of all reviews, or view each review individually. When a user selects a campsite, all the reviews for that campsite are shown on the campsite information page.
